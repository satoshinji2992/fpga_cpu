## ✅ 已修复的问题

### 1. 寄存器堆读取时序
**状态**: 已改为异步读取、同步写入，满足单周期CPU数据通路要求。

**当前实现**:
```verilog
assign rs1_data = (rs1 == 5'b0) ? 32'b0 : regfile[rs1];
assign rs2_data = (rs2 == 5'b0) ? 32'b0 : regfile[rs2];
```

### 2. 分支指令条件判断
**状态**: 已使用有符号/无符号比较分别实现BLT/BGE/BLTU/BGEU。

### 3. RV32I访存细节
**状态**: 已补齐LB/LH/LBU/LHU符号扩展与零扩展，SB/SH按地址偏移生成字节使能。

### 4. JALR地址对齐
**状态**: 已按RISC-V规范清除目标地址最低位。

### 5. ADDI负立即数
**状态**: 已修复ADDI在立即数高位为1时被误译码为SUB的问题。
